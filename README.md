# Quantitative Trading Strategy Development Project

## Project Overview
This project demonstrates the development of a complete quantitative trading system using NIFTY 50 data. It covers:

- Data acquisition and cleaning for Spot, Futures, and Options
- Feature engineering including EMA indicators, Greeks, IV, and derived metrics
- Regime detection using Hidden Markov Models
- Baseline trading strategy implementation with 5/15 EMA
- Machine Learning enhancements (XGBoost and LSTM) for trade signal filtering
- High-performance trade analysis and outlier detection
